# Secretos Arcanos

**Aceleración de Hechizo:** Cambias el tiempo de lanzamiento de un hechizo a una acción adicional. Si el hechizo es de daño, debes usar este efecto antes de realizar la tirada de ataque.

**Eco Arcano:** Cuando realizas una tirada de ataque para un conjuro y fallas, puedes gastar el coste correspondiente para volver a tirar el d20, pero tendrás que usar el nuevo resultado.

**Duplicación de Hechizo: A**fectas a un segundo objetivo con un hechizo de un solo objetivo. Si el hechizo es de daño, debes usar este efecto antes de realizar la tirada de ataque.

**Prolongación de Hechizo:** Duplicas la duración de un hechizo. Si el hechizo es de daño, debes usar este efecto antes de realizar la tirada de ataque.

**Extensión de Alcance:** Duplicas el alcance de lanzamiento de un hechizo. Si el hechizo es de daño, debes usar este efecto antes de realizar la tirada de ataque.

**Sobrecarga Arcana:** Aumentas el coste del hechizo a la mitad para aumentar los dados de daño del hechizo a la mitad. Debes usar este efecto antes de realizar la tirada de ataque.
